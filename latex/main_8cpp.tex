Main function that sets up and runs the load balancer simulation. 

Initializes a set number of web servers and generates a random set of requests. The \doxylink{class_load_balancer}{Load\+Balancer} then processes these requests over a specified duration of time.

\begin{DoxyReturn}{Returns}
int Returns 0 upon successful execution of the simulation. 
\end{DoxyReturn}
Loop to initialize the web servers.

Each server is initialized with an ID and a pointer to the log file for logging activities. The servers are then added to the priority queue.

Loop to generate and add random requests to the request queue.

Each request is given random input and output IP addresses and a random processing time. Requests are logged in the log file.
