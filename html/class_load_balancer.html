◆ addRandomRequests()

void LoadBalancer::addRandomRequests ( )

Adds a random request to the request queue at intervals.

A new request is generated with a random IP address and processing time. This occurs with a 20% probability.

◆ distributeRequests()

void LoadBalancer::distributeRequests ( )

Distributes requests to the available servers in the load balancer.

Attempts to assign each request in the request queue to the server with the least processing time. If a server cannot accept a request due to a full queue, the request is retained for later processing.

void LoadBalancer::logFinalStatus ( )

Logs the final status of the load balancer after the simulation ends.

This includes the remaining requests in the queue, the status of each server, and the number of discarded requests.
