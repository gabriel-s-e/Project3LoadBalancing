void WebServer::addRequest ( const Request & req)

Adds a request to the server's job queue.

If the server can accept the request, it adds it to the job queue and increments the current load. If the queue is full, the request is rejected and logged.

Parameters
reqThe request to be added to the server's queue.

◆ processJobs()

void WebServer::processJobs ( )

Processes jobs in the server's job queue.

If the server has jobs in its queue, it will process the job at the front of the queue, decrementing its processing time and current load. If the job is completed, it will be removed from the queue.
